## 🚀 GŌKU LAB - Plataforma de Evaluación

Plataforma integral para evaluar inteligencias múltiples y actitud emprendedora.

## ✨ Características

- 🧠 Test de Inteligencias Múltiples (56 preguntas)
- 🚀 Test de Actitud Emprendedora (10 atributos)
- 📊 Resultados detallados y visuales
- 🔍 Búsqueda y consulta de resultados
- 📱 Diseño responsivo

## 🛠️ Tecnologías

### Frontend
- React 18 + Vite
- React Router v6
- Framer Motion
- Chart.js

### Backend
- Cloudflare Functions
- MongoDB Atlas

```bash
